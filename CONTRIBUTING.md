# Contributing Guidelines

## Willkommen bei OpenManus PenTest!

Wir freuen uns über Ihre Beiträge zu unserem Projekt. Diese Richtlinien sollen Ihnen helfen, effektiv mitzuwirken.

## 🎯 Code of Conduct

- Respektvoller Umgang mit allen Community-Mitgliedern
- Professionelle Kommunikation
- Konstruktive Kritik
- Keine Diskriminierung oder Belästigung

## 📝 Pull Request Prozess

1. Forken Sie das Repository
2. Erstellen Sie einen Feature-Branch (`git checkout -b feature/AmazingFeature`)
3. Committen Sie Ihre Änderungen (`git commit -m 'Add some AmazingFeature'`)
4. Pushen Sie zum Branch (`git push origin feature/AmazingFeature`)
5. Öffnen Sie einen Pull Request

## 💻 Entwicklungsrichtlinien

### Code-Stil
- PEP 8 Standards befolgen
- Docstrings für alle Funktionen und Klassen
- Typ-Annotationen verwenden
- Aussagekräftige Variablennamen

### Tests
- Unit Tests für neue Funktionen
- Integration Tests für neue Features
- Bestehende Tests nicht brechen

### Dokumentation
- README.md aktualisieren
- Neue Features dokumentieren
- API-Dokumentation pflegen

## 🔒 Sicherheitsrichtlinien

- Keine sensiblen Daten im Code
- Sichere Konfigurationsoptionen
- Dokumentierte Sicherheitsmaßnahmen
- Responsible Disclosure für Schwachstellen

## 📚 Commit-Nachrichten

- Klare, beschreibende Nachrichten
- Präfix für Commit-Typ:
  - feat: Neue Features
  - fix: Bugfixes
  - docs: Dokumentation
  - style: Formatierung
  - refactor: Code-Refactoring
  - test: Tests
  - chore: Wartung

## 🤝 Review-Prozess

- Code-Review durch mindestens einen Maintainer
- Automatische Tests müssen bestehen
- Dokumentation muss aktuell sein
- Keine Breaking Changes ohne Diskussion

## 📞 Kommunikation

- Issues für Bugs und Feature-Requests
- Pull Requests für Code-Änderungen
- Diskussionen für Ideen und Vorschläge
- Security Policy für Sicherheitsprobleme

## 📋 Checkliste für Pull Requests

- [ ] Tests hinzugefügt/aktualisiert
- [ ] Dokumentation aktualisiert
- [ ] Code-Stil eingehalten
- [ ] Keine sensiblen Daten
- [ ] Commit-Nachrichten formatiert
- [ ] Branch aktuell mit main
